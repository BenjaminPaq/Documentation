### Introduction to WordPress
### WordPress Essentials: Domains & Hosting
- Domains are the unique address visitors type into their browsers to find a site
- Host is a service that stores a websites files and makes them accessible to the internet
- Hosting companies can be changed as needs arise – in many cases without changing the domain
- Bunding hosting and domains offer both hosting services and domain registration in one package
	- Simplifies the domain and hosting process
	- Avoids having to manage different accounts with different providers
- Domain names can be purchased separately before signing up with a hosting company
	- Gives more control over the domains setting
	- Makes it easier to switch hosting providers
- Purchasing and registering a domain follows a simple five-step process
	- Select a domain registrar – a company that manages domain name registration 
	- Search and select a domain name (can be country specific)
	- Register and configure domain settings
	- Choose a hosting provider
- When selecting a hosting provider, consider certain factors:
	- Reliability – providers with good uptime record and reliable customer support
	- Scalability – providers that offer easy hosting upgrades in line with site growth
	- Security – providers should have SSL certificates and regular backups
	- User Friendly Interface – Intuitive and easy to use control panels for easy site management
	- Affordability – ensure provider offers the above-mentioned features within defined budgets
- One-click hosting solutions will register a domain, install WordPress and host the site in a single click

### Choosing & Installing a Theme
- Themes are website templates with certain layout and design
- WordPress content is completely separated from presentation
	- The look of a site can change completely by swapping a theme while content remains the same
- Sites will not look the same as thumbnail or example pages
	- Often time requires specific plugins to achieve the initial appearance
- Thousands of themes are available for free in WordPress – free themes offer premium packages
- Themes are also available for download outside of the WordPress theme repository
- There are four categories of themes available:
	- Block Themes (future of WordPress) – Uses blocks for all parts of the site
		- Includes menus, header, footer, and content
		- From the site editor, pages can be created/edited. Sitewide styles can also be customized
		- Site editor allows for setting up or creating new templates
		- Block themes provide greater flexibility for designing and customizing the website
	- Classic themes – Do not use block editor to manage the site layout beyond posts and pages
		- Uses customizer, menus and widgets to make changes to the theme settings
	- Hybrid themes – Classic theme that adopts some features of site editing, such as template editor
	- Universal themes
		- Theme that can be configured as a block or classic theme
- Block and classic themes are the most commonly used on WordPress sites
- New themes can be added under Appearance  Themes
	- Selecting a theme will provide more details about a theme
	- Themes can be filtered using the feature filter
	- When reviewing themes, consider looking at version updates, active installations, and ratings
- Installing themes can be done in Appearances  Themes  Add New
	- Installed themes need to be activated in order to work on the site
	- Themes can be installed from zip file manually from the Upload Theme button
- Theme update notifications are available in the top menu bar and dashboard
- The Updates tab in the left sidebar contains all theme / plugin updates in a central location
	- Always back-up the site before making any theme or plugin updates
- Child theme is a sub-theme that inherits the look, feel, and function of the main theme
	- Child themes allow for upgrades to the parent theme without affecting customizations
		- Customizations are kept separate from the main theme’s files
- Themes can also be updated under Appearance  Themes

### Choosing & Installing Plugins
- Plugins are used to add extra functionality to a WordPress site
- Plugins can be found in the Plugins section from the left sidebar menu
- Select Plugins  Add New to search the plugin repository from WordPress
	- Consider the star rating, last update, active installations, and compatibility
	- Further details about plugins can be found by selecting details
- Plugins are first installed then activated through the plugin repository
- Plugins can also be added from local zip files
	- This is common for many premium plugin versions
- Plugins can be deactivated through the same plugin menu by selecting Deactivate
	- Once deactivated, plugins can be deleted entirely or reactivated if the deactivation is temporary
- Plugins may cause conflicts on sites – there are two ways of diagnosing plugin conflicts
	- Manually deactivate plugins and test them individually to find the problematic plugin
	- Use a troubleshooting plugin to diagnose plugin issues
