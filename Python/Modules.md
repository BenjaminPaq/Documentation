# Modules
Definition: Modules are pre-coded scripts used for specialized coding activities or functions
- All Python modules are contained within a .py file extension
- Modules commonly contain a series of functions, attributes and other modules
- Modules contain documentation that can be used to guide how they're used
- Modules can either be external files from the web or local files within the Python package
- Modules can be created locally by saving one or more functions to a .py extension

## Importing Modules
- Modules are imported using the import keyword followed by the name of the module
- Importing by module name imports the entire module and all of its associated script
- It is best practice to always import modules at the very top of the code
- Importing full modules can slow down code execution and processing times
- Individual functions can be imported by incorporating the from keyword
- The from keyword specifies the module while the import keyword specifies the function
- Multiple functions can be imported by separating them with commas
- Modules can be aliased or renamed at time of import to shorten their callback and execution

## OS Module
Definition: Module used to interact with the operating system
- ChDir: Changing the path of a directory
- Environ: Returns an informational directory
- GetCWD: Returns the current directory

## String Module
Definition: Module used to perform string operations

## Datetime
Definition: Module used to create and work with date objects
- Datetime: Function used to create a date using a year, month and date parameter
- Strftime: Formats a Datetime object into a readable string
	- 28 format directives exist for specific date formats

## Math
Definition: Module used to perform math operations using a simplified syntax
- Ceiling: Returns the rounded up whole number of the passed value
- E: Returns the digits of E
- Floor: Returns the rounded down whole number of a passed value
- Pi: Returns the digits of Pi
- Square Root: Returns the square root of any passed number
