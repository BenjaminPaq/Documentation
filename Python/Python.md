# Python
- Python is a general-purpose programming language used to build computer software.
- Python is a free to use open-source programming language
- Python code can be easily shared among developers through packages
- Python syntax closely resembles natural language, increasing its readability
- Python is a common first language to learn due to its ease of use and vast functionality
- Python Shell (Console): Executes code directly through a terminal
- Scripts: Contains code for execution
- File Structure: Python scripts are generally contained within a single root folder 
- Saving Scripts: Python scripts are saved using a a .py extension (script.py)
- Commands: A series of Python code instructions contained within a document
- Terminating Script: Pressing control + c on windows will terminate a session

## Use Cases
- Task automation
- Web applications
- Artificial intelligence and machine learning tasks
- Data analysis
- Data visualization and data reporting
- Performing basic and advanced calculations

## Development Pitfalls
- Python executes code closer then other similar programming languages like Java or C
