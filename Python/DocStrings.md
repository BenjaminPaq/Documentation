# DocStrings
Definition: Docstrings are blocks of text embedded inside a function that describe what that function does
- Display additional information about a function to help with the functions use
- Docstrings are classified as an attribute of a function
- Docstrings include a function overview along with information on arguments and returns
- Built-in docstrings are called on a function name as a method
- Custom docstrings can be initialized using triple string quotes
- Custom docstrings can either be a single line condensed string or multi-line
